# Sevens Token Smart Contracts

**Enterprise-grade NFT marketplace infrastructure built on Solana blockchain using Anchor framework**

## Overview

A comprehensive token ecosystem featuring unique hash-validated NFTs with built-in marketplace functionality. This project demonstrates advanced Solana development patterns and enterprise-grade smart contract architecture.

## Getting Started

### Prerequisites
```bash
Solana CLI 1.16+
Anchor 0.29+
Node.js 18+
Rust 1.70+
```

### Development Setup
```bash
git clone https://github.com/anatolii-semochko/sevens-smartcontracts.git
cd sevens-smartcontracts
npm install
anchor build
anchor test
```
